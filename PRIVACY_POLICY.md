# Privacy Policy for POLY PRISM

**Last Updated: January 2026**

## Overview

POLY PRISM is a Chrome extension that analyzes Polymarket prediction markets. This privacy policy explains what data we collect and how we use it.

## Data Collection

### Data We Collect

1. **User Preferences (Stored Locally)**

   - Extension enabled/disabled state
   - Auto-hide delay setting (5-120 seconds)
   - Activity timeframe preference (1h or 24h)

   This data is stored using Chrome's sync storage API and syncs across your Chrome browsers if you are signed into Chrome. We do not have access to this data on any server.

2. **Temporary Cache Data**
   - Market data from Polymarket APIs is temporarily cached in memory
   - Cache is cleared when you close your browser
   - No cache data is sent to any external server

### Data We Do NOT Collect

- Personal information (name, email, address)
- Browsing history outside of Polymarket
- Financial information or trading activity
- Authentication credentials
- Any data that identifies you personally

## Third-Party Services

This extension communicates with Polymarket's public APIs:

- gamma-api.polymarket.com (event and market data)
- clob.polymarket.com (orderbook data)
- data-api.polymarket.com (trade history)

These requests are made directly from your browser to Polymarket. We do not proxy, log, or store any of this data on our servers. Please refer to Polymarket's privacy policy for their data practices.

## Data Storage

- **Chrome Sync Storage**: Your preferences sync across devices via your Google account
- **Chrome Local Storage**: Rate limit state stored locally only
- **Memory Cache**: Temporary market data cleared on browser close

## Data Sharing

We do not share, sell, or transfer any user data to third parties. All data remains on your device or in your Chrome sync storage.

## Your Rights

You can:

- Clear all extension data by uninstalling the extension
- View stored data via Chrome DevTools (Application > Storage)
- Disable sync storage by using Chrome in guest mode
